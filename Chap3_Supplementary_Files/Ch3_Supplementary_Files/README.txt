# FigTree Files 

Instructions to open FigTree files: 
1. Download FigTree software from http://tree.bio.ed.ac.uk/software/figtree/ 
2. Open the FigTree file using the downloaded software. 
3. To colour the sequence tips by their PhyCLIP cluster annotation, collapse the "Tip Labels" menu. 
4. Under "Colour by:", select "CLUSTER". Change the colour setup by clicking on the "Colour" next to "Setup:" in the same menu.
5. A pop-up will appear. Drop down the "Scheme:" menu and select "Fixed Colours".
